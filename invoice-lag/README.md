# invoice-lag — 発注書PDF → 請求書Excel 自動生成スキル

発注書（PDF）をアップロードするだけで、請求書Excelを自動生成します。  
**請求書テンプレートは SKILL.md に内蔵済みのため、ユーザーはテンプレート不要です。**

## できること

- 発注書PDFから以下を自動抽出・転記
  - 管理No、担当者名、件名、発注日、納品日
  - 明細（内容・数量・単価）最大29件
- 金額・消費税・合計はテンプレートの数式が自動計算

## インストール方法

```
/mnt/skills/user/invoice-lag/
└── SKILL.md   ← このファイルだけ配置すればOK
```

## 使い方

Claude に発注書PDFをアップロードして、以下のように話しかけるだけです：

- 「請求書を作成して」
- 「発注書から請求書に転記して」
- 「自動入力して」

## 必要なライブラリ

```bash
pip install pdfplumber openpyxl --break-system-packages
```

## 対応フォーマット

特定の発注書フォーマット（管理No・担当者・件名・指定納期・明細の構成）に対応しています。  
フォーマットが異なる場合は `SKILL.md` 内の正規表現パターンを調整してください。
